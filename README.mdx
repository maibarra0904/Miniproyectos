# 1.ProgramaManejoDeNombres

Programa que recibe los nombres de 3 integrantes de tu familia o amigos como argumentos.

Se pide por cada nombre de la persona una nueva variable del tipo String al tomar el segundo carácter pero convertido en mayúscula y se le concatena un punto y los dos últimos caracteres de la persona. Por ejemplo para Andres debe quedar como N.es

Debe imprimir como resultado los tres nuevos nombres separado con guion bajo (como una única variable).

Ejemplo, un resultado final esperado para los nombres Andres, Maria y Pepe podría ser:

N.es_A.ia_E.pe

# 3.ProgramaEstanqueGasolina

Programa que suponen un estanque de gasolina (gas) con capacidad 70 litros y pida la medida actual en litros y muestra el resultado de la forma:

Si la capacidad actual es 70 litros: imprime Estanque lleno

Si está entre 60 y menor a 70: imprime Estanque casi lleno

Si está entre 40 y menor a 60: imprime Estanque  3/4

Si está entre 35 y menor a 40: imprime Medio Estanque 

Si está entre 20 y menor a 35: imprime Suficiente

Si está entre 1 y menor a 20: imprime Insuficiente
